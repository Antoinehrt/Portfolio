Internship

Fullstack Developer at ETNIC Brussels (02/2024 - 05/2024)



During my internship at ETNIC's Innovation Center (R&D department), I worked as a Fullstack Developer to design and develop an internal web application aimed at document management and interactive chatbot integration. The key feature of this application was its ability to act as a Retrieval-Augmented Generation (RAG) system, allowing users to interactively query and discuss documents stored within the platform. The primary goal was to streamline document-related tasks and enable intuitive access to information, revolutionizing how employees interacted with organizational knowledge.
Key Contributions
Throughout this internship, I collaborated closely with a small team of developers, working from the ground up to build a seamless and user-friendly application. My contributions included:

  • Frontend Development: The user interface is built using Angular 17, TypeScript, Angular Material, HTML, and SCSS, ensuring an intuitive and accessible user experience.
  • Backend API Implementation: Developed the backend REST API with Python and FastAPI, ensuring robust performance and scalability.
  • Asynchronous Task Management: Designed an efficient file upload system using Celery and Redis to handle long-running tasks asynchronously. This improved the system's performance and reliability.
  • Chatbot Integration: Implemented a conversational AI chatbot using a RAG architecture powered by LangChain, generative AI (Mistral AI), and vector search. This allowed users to receive contextualized responses based on document content.
  • Database Management: Managed data storage with MongoDB Atlas for the main database and Redis for caching and task queuing.
  • Testing and Quality Assurance: Wrote and executed tests using Pytest to ensure code reliability and maintainability.
  • Containerization and Deployment: Utilized Docker for containerized development, ensuring consistent environments across local and cloud-based systems on Azure.
Project Methodology
We followed the Agile methodology, working in two-week sprints. The team actively used stories, sprint planning, and retrospectives to ensure iterative progress and clear communication. This collaborative environment helped me gain firsthand experience in professional software development workflows.
Takeaways and Growth
This internship was a transformative experience that allowed me to Strengthen my skills in fullstack development by working on a complete project lifecycle. I also explored the integration of generative AI into real-world applications and Deepen my understanding of asynchronous processing and task queues for performance optimization. This project allow me to enhance my teamwork and project management abilities in an Agile environment.
By the end of the internship, I had gained invaluable insights into designing innovative solutions and integrating cutting-edge technologies into practical use cases
