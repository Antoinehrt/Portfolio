University College

Computer Science, specializing in Application Development HELHa Mons (2021 - 2024)



I completed my Bachelor's degree in Computer Science, specializing in Application Development, at HELHa in Mons. The program trained me as a Fullstack Developer, equipping me with the ability to work on both frontend and backend systems. However, through my projects and practical experiences, I gained more extensive expertise in backend development, where I excel at designing and managing robust systems.
Notable Projects
Throughout my studies, I participated in several impactful projects that prepared me for professional challenges:

  • Web Application and APIs: I developed scalable web applications and a mobile application using modern frameworks and tools, integrating APIs and databases into functional, scalable systems. One project used Angular for the frontend, C# (ASP.NET Core) for backend APIs, and SQL Server for database management. The other was developed using Bootstrap for the frontend part and Jakarta EE to create the backend APIs, These projects involved containerization with Docker, ensuring smooth deployment and scalability.
  • Game Development: I created video games using C++ and JavaFX, exploring graphical rendering, game mechanics, and performance optimization. These projects helped me understand low-level programming and interactive system design.
  • Mobile Applications: I built a mobile application using Kotlin and Android Studio, focusing on user experience and seamless integration with backend APIs.
  • Networks and Systems: Learned technical architecture and hardware systems, along with network fundamentals.
  • Websites: Developed web platforms with PHP, leveraging database systems like MySQL and Amazon SQL to ensure efficient data handling and secure user management.
  • Real-World Scenarios: Every project followed a complete software development lifecycle, including requirements gathering, design (with UML and wireframes), coding, testing, and deployment. Additionally, I mastered Scrum and Agile methodologies, regularly applying them to project management, ensuring clear communication, iterative progress, and delivery of high-quality results. These projects often required self-learning of new tools and technologies to meet specific objectives.
Academic Focus

The program offered a mix of technical and theoretical courses that broadened my understanding of computer science:

  • Software Development and Design: I explored design patterns, system architecture, and best practices for robust and maintainable software.
  • Mathematics and Machine Learning: I gained a foundation in analytical problem-solving and data-driven systems.
  • Technical Systems and Networks: I studied the fundamentals of networking, hardware, and operating systems, enabling me to understand how software interacts with underlying infrastructure.
  • Business and Communication: Courses in microeconomics, macroeconomics, and English for Computer Scientists enhanced my ability to communicate and collaborate in a professional context.

This project was part of my internship, detailed in its respective section.
