After high school, I pursued a Bachelor's in Computer Science at the École Polytechnique de Louvain in Louvain-la-Neuve, studying there for two years. This program created my understanding of programming, algorithms, and the theoretical foundations of computer science and deepened my passion for new technologies.
  • Programming Courses: Developed strong skills in Python and Java through courses and projects, including the creation of a web application using Python, Flask and HTML or Java application with Unittest and a quiz application in Python. In my second year, I independently learned C to complete a team project.
  • Algorithms and Paradigms: Studied algorithm design and explored different programming paradigms, including concurrency.
  • Foundations of Computer Science: Gained insights into computability, logic, and complexity, as well as the hardware basics of computer science.
  • Language: Developed technical communication skills in English for Computer Scientists.
Although I didn’t complete the program, these courses gave me a solid technical foundation, practical experience, and the ability to adapt quickly to new tools and concepts.
